<h1>3N+1 problema</h1>

<p>
3N+1 problema, dar žinoma kaip Collatz problema, yra matematinė problema,
kurioje nagrinėjama seka, apibrėžta taip:
</p>

<ul>
  <li>Jei skaičius yra lyginis, jis dalijamas iš 2</li>
  <li>Jei skaičius yra nelyginis, jis dauginamas iš 3 ir pridedamas 1</li>
</ul>

<p>
Procesas kartojamas tol, kol gaunamas skaičius 1.
</p>
